# GetuiSDK
========

## 个推SDK

### 安装方法

在 `Podfile` 文件下添加

``` pod 'Getui' ```

欢迎大家提交Issuse或者PR来提醒更新  

该版本使用了IDFA，请开发者提交审核时注意阅读文档。如果无需使用IDFA，请使用[Getui-NoIDFA](https://github.com/fmricky/Getui-NoIDFA)版本
